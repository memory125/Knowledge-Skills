## 🎭 生活化比喻

### 就像...

想象一个教室里有 50 个学生：

**传统银行方式：**
```
学生 A → 把账交给班长（银行）
班长记账 ✅
只有班长的账本有效
如果班长改账？大家看不出来 ❌
```

**区块链方式：**
```
学生 A → 大声说"我转给 B100 块"
全班 50 人都在自己的本子上记下 ✅
如果有人想改账：
  "等等！我本子上不是这样写的" 🙅‍♂️
  其他 49 人也不同意 ❌
修改失败！
```
