# 📚 Ebook Analyzer - 电子书智能分析工具

一个完全本地化的电子书分析工具，支持多种格式，无需外部 API，利用本地 Ollama 模型进行深度内容分析和总结。

## 🎯 核心功能

### 1. 多维度内容分析
- **本书内容**：完整章节结构、主要内容概述
- **一句话总结**：精炼概括全书核心
- **逻辑结构图谱**：非小说类的概念关联图 / 小说类的人物关系图
- **本书要点**：分章节/主题的核心观点提取
- **金句汇总**：最有价值的名言警句
- **作者思想**：核心论点、价值观、世界观
- **现实意义**：对当下生活的启发和实用价值
- **适合读者**：目标人群画像
- **书籍评分**：综合评估（内容深度、可读性、启发性）

### 2. 支持的格式
| 格式 | 状态 |
|------|------|
| PDF | ✅ 支持 |
| EPUB | ✅ 支持 |
| MOBI | ⚠️ 需转换 |
| DOCX/DOC | ✅ 支持 |
| PPT/PPTX | ✅ 支持（大纲模式） |

### 3. 输出格式
- Markdown 笔记（默认）
- JSON 结构化数据
- 文本摘要

## 🚀 使用方法

```bash
# 基本用法
ebook-analyzer analyze /path/to/book.pdf

# 指定输出格式
ebook-analyzer analyze /path/to/book.epub --format markdown

# 只提取特定维度
ebook-analyzer extract /path/to/book.pdf --type "金句，要点"

# 导出为 JSON
ebook-analyzer analyze /path/to/book.pdf --output ./notes.json
```

## ⚠️ 重要原则

1. **本地优先**：使用 Ollama 本地模型，无需外部 API
2. **隐私保护**：书籍内容不会离开本地环境
3. **格式友好**：支持常见的电子书和文档格式
4. **深度分析**：不止是总结，更注重理解和启发
