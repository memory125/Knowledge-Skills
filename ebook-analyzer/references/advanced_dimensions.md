## 🔬 扩展维度（新增）

### A. 文本结构层

#### 10. 写作手法分析
- **叙事技巧**: 第一人称/第三人称、时间线、视角转换
- **修辞手法**: 比喻、排比、对比、讽刺等使用频率
- **语言风格**: 简洁、华丽、幽默、严肃、诗意等
- **结构安排**: 悬念设置、伏笔呼应、节奏控制

**适用**: 小说、散文、文学作品

---

#### 11. 章节脉络图
```
第 1 章：铺垫 → 第 2 章：转折 → 第 3 章：高潮 → ...
```
- 每章核心任务
- 章节间逻辑连接
- 情节/论点的递进关系

**适用**: 所有类型书籍

---

#### 12. 关键词云
提取全书最高频的 50-100 个词，生成概念热力图

```
高频词统计:
1. 自由 (156 次)
2. 责任 (134 次)
3. 真理 (98 次)
...
```

**适用**: 哲学、社科类书籍

---

### B. 思想内容层

#### 13. 核心问题清单
列出本书试图回答的 3-5 个根本问题：
```
Q1: 什么是真正的幸福？
Q2: 如何在困境中保持希望？
Q3: ...
```

**适用**: 哲学、心理学、自我成长类

---

#### 14. 概念词典
提取并定义本书的 10-20 个核心术语：
```
[深度工作] 定义：在无干扰状态下进行职业活动...
[原子习惯] 定义：微小的改变带来巨大成果...
```

**适用**: 专业书籍、学术著作

---

#### 15. 论证逻辑链（非虚构类）
```
前提 1 → 证据 A → 推论 1
前提 2 → 证据 B → 推论 2
↓
最终结论
```

评估：
- 前提是否成立？
- 证据是否充分？
- 推理是否严密？
- 结论是否有必然性？

**适用**: 议论文、学术著作、商业书籍

---

#### 16. 人物弧光分析（小说类）
追踪主要人物的成长轨迹：
```
主角 A:
起点 → 遭遇挑战 → 内在转变 → 最终状态

性格变化:
- 初始特质
- 触发事件
- 关键时刻的选择
- 最终收获
```

**适用**: 小说、传记

---

#### 17. 冲突类型分析
```
内部冲突：理想 vs 现实
外部冲突：个人 vs 社会
关系冲突：友情/爱情/家庭
生存冲突：人 vs 自然/命运
```

**适用**: 小说、戏剧

---

### C. 知识体系层

#### 18. 知识图谱
绘制本书涉及的知识网络：
```
中心概念 → 相关理论 → 具体方法 → 案例应用
```

**适用**: 专业知识类书籍

---

#### 19. 方法论清单
提取可实操的方法步骤：
```
方法 1: [名称]
- 第一步...
- 第二步...
- 第三步...
- 关键注意点...
```

**适用**: 工具书、技能提升类

---

#### 20. 案例库
摘录本书中的典型案例：
```
案例 1: [简述]
- 背景
- 问题
- 解决过程
- 结果
- 启示
```

**适用**: 商业、管理、教育类

---

### D. 批判性思维层

#### 21. 潜在偏见识别
分析作者可能的立场和局限：
- **时代局限**: 写作年代的认知边界
- **文化偏见**: 特定文化视角的盲点
- **个人经历**: 作者背景带来的选择性关注
- **利益关联**: 赞助、推广等潜在影响

**适用**: 所有书籍，特别是观点类

---

#### 22. 逻辑漏洞检测
```
发现的问题:
1. 概念偷换：...
2. 因果倒置：...
3. 以偏概全：...
4. 诉诸权威：...
5. 非黑即白：...
```

**适用**: 议论文、社科类

---

#### 23. 反方观点
思考"如果反对这本书的观点，会怎么说？"

**适用**: 培养独立思考能力

---

#### 24. 争议点整理
列出学界/读者对本书的主要争议：
- 支持方理由
- 反对方理由
- 中立评价

**适用**: 有争议的书籍

---

4. ...
